Parcial API - Fórmula 1

Esta API combina información en tiempo real de OpenF1 con una base de datos propia en MongoDB.

Rutas principales:

  • /api/drivers → Listado de pilotos (OpenF1)
  • /api/favorites → CRUD de favoritos (MongoDB)
